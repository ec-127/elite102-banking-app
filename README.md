# elite102-banking-app

# Project Description
A terminal-based banking app created with Python and MySQL. It allows users to create accounts, deposit, withdraw, and check balances. 

# Setup Instructions
1. Install Python
2. Install SQL
3. Install connecter: pip install mysql-connector-python
4. On MySQL run: CREATE DATABASE banking_app;
5. Open main.py and update password with your own MySQL password
6. Run the program
